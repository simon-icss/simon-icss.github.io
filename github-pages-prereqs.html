Learn to Code (Build a Webpage)

Steps To Take Before The Meetup

You will need a GitHub account. If you don't have one already, sign up for one:
https://github.com/

Your GitHub username will be part of the URL of your webpage, and you'll be eventually using your GitHub profile as a way to show your work to potential employers, so choose your username carefully.

Download and install a text editor. Personally, I like Atom:
https://atom.io/

But if you run into issues installing Atom, or already have another editor that you prefer, any of these will work:
http://brackets.io/
https://code.visualstudio.com/
https://notepad-plus-plus.org/ (this is a good choice for Windows users)
https://www.sublimetext.com/ (this one has an occasional nag screen wanting you to pay for it, but you can just cancel out of it when it comes up)

Download and install GitHub Desktop (Windows or Mac):
https://desktop.github.com/

If you run into issues installing GitHub Desktop on Windows, or you'd just prefer to use Git from the command line and follow along with the command line instructions during the Meetup, there's another option. You can install this instead, keeping the default settings checked during the install process:
https://git-for-windows.github.io

Note that using Git from the command line is far less friendly to beginners and requires more steps to get the same things done--but we will be coming around to assist each person regardless of their setup.

Once you have these set up, you should be ready to follow along with the session during the meetup!
